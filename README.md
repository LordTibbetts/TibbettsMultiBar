# TibbettsMultiBar
**Tibbetts MultiBar** is a lightweight, fully customizable replacement for the default XP and Reputation bars, designed to work reliably across Retail, Classic Era, TBC Anniversary, and Wrath clients.

It focuses on visual clarity, clean configuration, and cross-version stability, avoiding fragile UI features that break between WoW versions.

**Features:**

Custom XP and Reputation bars  
Texture and font selection with live preview  
Universal texture picker with:  
visual previews  
searchable lists  
texture names overlaid on the preview

**Optional tick presets:**

No ticks  
Every 10%  
Every 20%  
Optional percentage tooltip when hovering ticks  
Blizzard-style color pickers with opacity slider  
Supports separate settings for XP and Reputation  
Minimal CPU usage, no combat overhead  
No SharedXML dependencies.  
No client-specific UI hacks.

**Configuration:**  
/tmp  or,  
via Interface → AddOns → Tibbetts MultiBar

**Settings include:**  
Bar size, position, and scale  
Textures and fonts  
Tick presets  
Colors and opacity  
XP and Reputation options independently

**Design Goals:**  
Clean, readable bars  
Simple presets instead of fragile sliders  
Stable behavior across all WoW clients  
Easy to configure, easy to maintain

**Notes:**  
Saved variables are profile-based  
Safe to enable/disable without UI reload loops  
Designed for players who want a modern bar without replacing their entire UI
